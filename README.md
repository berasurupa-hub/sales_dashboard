Sales Performance Analysis (Excel Dashboard)

Project Overview
This project analyzes sales data using Microsoft Excel to identify
sales trends, top-performing products, and regional performance.

Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- Data Cleaning

Dataset Features
The dataset includes:
- Order Date
- Product Name
- Region
- Quantity
- Sales

Dashboard Insights
The dashboard provides:

1. Sales by Region
Shows which region generates the most revenue.

2. Monthly Sales Trend
Displays sales patterns over time.

3. Product Sales Analysis
Identifies the top-performing products.

4. Total Sales KPI
Displays total sales value.
